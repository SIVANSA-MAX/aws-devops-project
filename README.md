# AWS DevOps Project

##  Project Overview
This is a beginner DevOps project where I deployed a simple website using Docker.

##  Tools Used
- Docker
- Nginx
- GitHub

##  What I Did
- Created a simple HTML website
- Containerized it using Docker
- Deployed using Nginx

##  How to Run
1. Clone the repository
2. Build Docker image:
   docker build -t my-app .
3. Run container:
   docker run -d -p 80:80 my-app

##  Output
Simple web page running inside Docker container
